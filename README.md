## **Installation**

### Prerequisites
- Node.js
- npm or yarn package manager

### Steps
1. Clone the repository:
```bash
   git clone https://github.com/balikis-oyeleye/atomic-landing-page.git
```
   
2. Go to the project directory

```bash
  cd atomic-landing-page
```

3. Install dependencies

```bash
  npm install
```

4. Start the project

```bash
  npm run dev
```
